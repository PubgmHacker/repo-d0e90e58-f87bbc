# AI Orb — шейдерная сфера ассистента для iOS

Живая «дышащая» сфера в стиле Siri / Салют / Алисы: меняет цвет, скорость и характер движения в зависимости от состояния ассистента.

## Почему шейдер, а не USDZ-модель

Статичная 3D-модель (USDZ/glTF) не умеет плавно перетекать между цветами и «жидкими» состояниями — такие орбы у Apple и Сбера рисуются процедурным шейдером в реальном времени. Этот вариант легче (один Swift-файл, без ассетов), работает на 60 FPS через Metal и выглядит одинаково на всех устройствах.

## Установка

1. Добавьте `AssistantOrbView.swift` в проект Xcode.
2. Никаких зависимостей и ассетов не нужно (шейдер компилируется из исходника при запуске).
3. Минимальная версия: iOS 15+.

## Использование

```swift
struct VoiceScreen: View {
    @State private var orbState: OrbState = .idle

    var body: some View {
        AssistantOrbView(state: orbState)
            .frame(width: 280, height: 280)
    }
}
```

Смена состояния — просто присваивание, переход анимируется автоматически (~0.5 с):

```swift
orbState = .listening  // Слушаю  — зелёно-бирюзовая, активная пульсация
orbState = .thinking   // Думаю   — янтарно-фиолетовая, густое перемешивание
orbState = .speaking   // Отвечаю — розово-синяя, быстрые волны
orbState = .idle       // Ожидание — сине-фиолетовая, спокойное дыхание
orbState = .error      // Не расслышал / ошибка — красно-коралловая, мягкая тревожная пульсация
```

## Реакция на голос (опционально)

Передайте уровень громкости микрофона или синтеза речи (0…1), и сфера будет пульсировать в такт:

```swift
AssistantOrbView(state: .listening, audioLevel: micLevel)
```

Уровень можно брать из `AVAudioRecorder.averagePower(forChannel:)` или tap на `AVAudioEngine`.

## Настройка под бренд

Цвета и характер каждого состояния задаются в `OrbState.params` (три цвета + скорость, искажение, пульсация). Меняйте hex-значения в одном месте.
