# Plink Launch Plan

**Version:** 1.0 · 2026-07-15  
**North star:** Category-defining watch-together app (CIS first), not a 30-day “50M downloads” miracle.

## Phase 0 — Closed beta (now → +2 weeks)

- TestFlight + Desktop + APK  
- Fix P0: Android sync, LiveKit or hide voice, IAP sandbox  
- Prove sync lab &lt;2s on 3 devices  
- Analytics: core funnel events  

**Exit:** BETA_TEST_PLAN exit criteria met.

## Phase 1 — Soft launch RU/CIS (+2 → +8 weeks)

### Product
- YouTube + VK + Rutube primary  
- Cinema services stay OFF on App Store builds  
- Plink+ live with clear free limits (server-enforced)  
- Voice on if LiveKit stable; else ship without mic  

### Growth
- Landing `plink.io` / current domain: DMG, EXE, APK, App Store  
- TikTok/Reels: 15s “sync reveal” + AI orb  
- Student / friend-group seeding (universities, dorms)  
- Referral: room invite deep links  

### Targets
| Metric | Soft launch target |
|--------|-------------------|
| MAU | 10k → 100k |
| D7 | ≥20% |
| Crash-free | ≥99.5% |
| Rating | ≥4.5 with &lt;500 reviews careful support |

## Phase 2 — Regional scale (+2 → +6 months)

- Android feature parity  
- Performance: API edge closer to users  
- Creator packs / seasonal emoji  
- Press: RU tech media, Product Hunt EN later  

## Phase 3 — Global attempt (+6 → +18 months)

Only if:
- D7 ≥25% and sync trust high  
- Legal path for multi-service (screen-share mode **or** partnerships)  
- CAC &lt; LTV in paid tests  

### Aspirational (not commit)
- 1M MAU / 12 months  
- 10M+ installs if viral loops work  
- 50–100M requires category win + multi-year distribution (Rave-class)

## App Store submission checklist

- [ ] Screenshots 6.7 / 6.5 / iPad  
- [ ] App Preview 15–30s  
- [ ] Privacy nutrition labels accurate  
- [ ] Review notes: test account, no Netflix claims  
- [ ] IAP products + screenshots of paywall  
- [ ] ENABLE_CINEMA false in release  

## Marketing messages (approved)

✅ “Watch YouTube, VK, Rutube in perfect sync”  
✅ “AI companion for your movie night”  
✅ “Living themes & custom emoji”  
❌ “Watch Netflix together” (App Store + ToS)  
❌ “Better than Discord servers” (scope trap)
